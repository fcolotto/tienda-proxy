# Política de Privacidad – María T y Ozone

Este asistente utiliza una API privada para consultar información de pedidos y productos de una tienda que opera en la plataforma Tienda Nube, con el único objetivo de responder consultas operativas realizadas por los usuarios.

## Datos que se procesan
El asistente puede procesar, cuando es necesario, los siguientes datos:
- Información de pedidos (estado, productos, envío, tracking).
- Datos básicos asociados a un pedido, como nombre, email o dirección de envío.

## Uso de la información
La información se utiliza exclusivamente para:
- Responder la consulta específica realizada por el usuario.
- Brindar información operativa sobre pedidos, productos o envíos.

Los datos **no se almacenan**, **no se reutilizan** y **no se comparten** fuera del contexto de la consulta.

## Seguridad
El acceso a la API está protegido mediante una clave de seguridad. Solo el asistente autorizado puede realizar consultas a la API.

## Terceros
Este asistente no vende, cede ni utiliza los datos para fines publicitarios, analíticos o comerciales externos.

## Contacto
Si el usuario desea atención humana o tiene dudas sobre el uso de datos, puede solicitar la derivación a un canal de atención humana.
